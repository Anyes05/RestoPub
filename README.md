# RestoPub - Sistema de Gestión para Restaurante

Proyecto académico desarrollado en C++ para la materia Programación Avanzada.

## Descripción

RestoPub es un sistema de gestión para restaurante orientado a la administración de productos, ventas, mesas, empleados y pedidos a domicilio.

El proyecto fue desarrollado aplicando conceptos de análisis y diseño orientado a objetos, utilizando diagramas UML y patrones de diseño para estructurar el sistema.

---

# Tecnologías Utilizadas

* C++
* Makefile
* UML
* Programación Orientada a Objetos

---

# Conceptos Aplicados

* Diagramas de Dominio
* Diagramas de Secuencia de Sistema
* Diagramas de Comunicación
* Diagramas de Clases de Diseño (DCD)
* Patrones de Diseño:

  * Factory
  * Singleton
* Encapsulamiento
* Polimorfismo
* Asociación y composición
* Arquitectura orientada a objetos

---

# Funcionalidades

* Alta y baja de productos
* Gestión de ventas
* Asignación de mesas
* Gestión de mozos y repartidores
* Ventas a domicilio
* Facturación
* Manejo de menús
* Carga de datos de prueba
* Interfaz por consola basada en roles

---

# Roles del Sistema

* Administrador
* Mozo
* Repartidor
* Cliente
